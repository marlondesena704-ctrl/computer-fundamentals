# Computer Fundamentals

Este repositório contém resumos e anotações sobre **fundamentos da computação**, organizados por tema.

Os conteúdos abordam conceitos importantes de:

- Hardware
- Redes
- Cloud Computing
- Sistemas
- Troubleshooting

Os resumos foram organizados a partir de uma playlist educacional sobre informática e infraestrutura de TI.

## Estrutura

hardware → Componentes físicos do computador  
redes → Funcionamento das redes  
cloud → Computação em nuvem  
sistemas → BIOS, partições e sistema  
troubleshooting → Diagnóstico de problemas
