# IP e DHCP

## Vídeo 27 — DHCP

DHCP atribui automaticamente:

- endereço IP
- gateway
- DNS

## Vídeo 28 — IP público e privado

IP público:

identifica a rede na internet.

IP privado:

usado dentro da rede local.

## Vídeo 26 — Firewall

Firewall controla o tráfego de rede.

Funções:

- bloquear acessos não autorizados
- monitorar conexões
