# Placa-mãe

## Vídeo 21 — O que é placa-mãe

A placa-mãe conecta todos os componentes do computador.

Componentes conectados:

- CPU
- memória RAM
- armazenamento
- placas de expansão

## Vídeo 22 — Diferença entre placa-mãe barata e cara

As diferenças incluem:

- qualidade do VRM
- quantidade de portas
- suporte a overclock
- suporte a mais armazenamento

## Vídeo 23 — Como escolher uma placa-mãe

Critérios importantes:

- compatibilidade com processador
- tipo de memória RAM
- slots de expansão
- suporte a SSD M.2
